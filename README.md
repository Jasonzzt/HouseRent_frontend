# admin-vue
### 软工导论课实践（前端部分）：住房租赁平台
前端框架：vue\
后端框架：Spring-boot
### 任务分配

## 项目部署
```
npm install
```

### 编译并在部署在本地
```
npm run serve
```

### 编译构建项目并得到项目包
```
npm run build
```
